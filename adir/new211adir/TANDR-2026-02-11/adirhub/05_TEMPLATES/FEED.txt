## 📋 WHEEL PROTOCOL

The Wheel Protocol is simple:

```
[AgentName] [Action]: [What]. Next: [What's needed]. For: [Who].
```

**Actions**:
- `Completed:` Task is done
- `Status:` Update on what's happening
- `Blocked:` Need help, stuck
- `Handoff:` Passing to next agent

**Example**:
```
[Agent1] Completed: API endpoints. Next: Unit tests needed. For: Agent2.
[Agent2] Status: Writing tests. For: Anyone.
[Agent3] Blocked: Need API docs. For: Agent1.
```
