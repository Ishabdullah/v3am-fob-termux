Troubleshooting Guide

  • post_max_size error: Your image + overhead exceeds the limit. Increase in php.ini or use smaller image.
  • Connection refused: PHP process crashed. Check upload log for errors.
  • Connection reset: Request was too large or took too long. Check file size < post_max_size.
  • Memory exhausted: Increase memory_limit in php.ini to 512M or 1G.
  • Timeout: Increase max_execution_time to 360+ seconds.
