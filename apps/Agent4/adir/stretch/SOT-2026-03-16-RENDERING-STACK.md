# SOT-2026-03-16-RENDERING-STACK  
## Discovery: Self-Served Iframe Bridge  
Agent4 confirmed that the chat UI strips embedded JS/CSS but permits iframes to external URLs.  
Since Agent4 is a Node+PHP server, it can serve its own HTML files.  
This allows full JS/Canvas applications to be rendered inside the chat via local port loopback.
