📝 How to Update THE PLAN

For AI Assistants:
  1. Create new entry file in entries/ directory with descriptive name and date
  2. Use the template from entries/_template.html
  3. Add new entry card to this index page (don't remove old ones)
  4. Update stats if applicable (audit logs count, apps documented, etc.)
  5. Preserve all existing entries - this is a GROWING system
⚠️ Important Rules:
• NEVER delete old entries
• ALWAYS use dated filenames (e.g., desktop-audit-2026-01-06.html)
• Each audit/update gets its own file
• Index page shows chronological entries
• Token efficiency: Reference specific entry URLs, not entire history
