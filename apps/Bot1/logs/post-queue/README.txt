# POST QUEUE FOLDER

This is where you drop post ideas for the AI to review and decide what to post.

## HOW TO USE:

1. Create a .txt file with your post idea (any name is fine)
2. The AI can read all files in this folder
3. Ask the AI: "Check the post-queue and tell me which posts are ready"
4. The AI will:
   - Read all posts in the queue
   - Analyze which ones fit the brand voice
   - Suggest which to post and when
   - Help refine posts that need work
   - Move approved posts to post-drafts/

## FILE FORMAT:

Simple text files work best. Example:

```
Title: Kitchen Remodel Tips
Platform: Facebook
Priority: Medium

Post Content:
🔨 Planning a kitchen remodel? Here are 3 things to consider BEFORE you start...
[rest of post]

Notes:
- Could use before/after photos
- Targets homeowners 35-55
- Educational tone
```

## WORKFLOW:

1. Drop post ideas here
2. Ask AI: "What's in the post-queue?"
3. AI reviews and suggests improvements
4. AI can post directly (with temp Graph Explorer token)
5. OR AI saves refined version to post-drafts/ for manual posting

## NOTES:

- Files stay here until you delete them
- AI won't auto-post without being asked
- Use Graph Explorer tokens when testing posting
- AI will help you decide what's ready to post
